SIQ UseCase - Data Remediation - Quarantining

Background

This use case provides the ability to demonstrate Data remediation functionality. In this case, having selected a specific Data Classification (Already discovered), a Data remediation policy, when run moves the files under the specified Data Classification to a QUARANTINE folder for investigation by a Compliance or Security Officer.

A tombstone file is left in the original place explaining why the file was moved and who to contact about it.
