*************
* Execution *
*************
Create a report from the template "Task Result Status Report" from: 

Intelligence --> Reports --> Identity and User Reports

Fill out the from the report, then run. 
************
* Background *
**************
Norm created this as a part of a POC. It is a report that shows the task results filtered by: 
- date range of when the task was completed; and 
- the status of the task (Error/Warning/Success/Terminated)

****************
* To Configure *
****************
Import via setup.xml 

******************
* To Demonstrate *
******************
1. Navigate to Intelligence --> Reports --> Identity and User Reports and find the "Task Result Status Report" report template. 
2. Follow the form and create a new report with a name you've chosen. 
3. Select the "Completion Status". You can choose one or multiple of the statuses and choose a date range. 
4. Configure the report layout in the next screen and you can click either "Save and Preview" for a preview or the report or "Save and Execute" 
for the report to run.
