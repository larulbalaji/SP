Graphical Dashboard Plugins
October 2017

Contact: patrick.sena@sailpoint.com

**************
* Background *
**************
These plugins are graphical dashboards with drill down capabilities on identities, accounts, roles and user entitlements. 
These are built based on the same js framework as the Identity Dashboard plugin (Mike Kiser).

****************
* To Configure *
****************
1 Install the plugins 
2 Home--> Edit --> Quick Links --> Add the desired dashboard Quick Link to the Home page --> Save. 


******************
* To Demonstrate *
******************
Click on the Quick Link for each of the dashboards: 
- Accounts Dashboard
- Identities Dashboard
- Roles Dashboard
- User Roles and Entitlements Dashboard 

NOTE: It may take a few seconds for each of the dashboards to load.
