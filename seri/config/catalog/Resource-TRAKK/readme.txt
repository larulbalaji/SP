* Background *
**************
TRAKK is a time-tracking read-only JDBC application used in the Fundamentals Training class.
This relies on the MYSQL tables in the SERI Resource VM.

****************
* To Configure *
****************
This resource is a part of the SERI Standard Demo. 
Therefore, the necessary tasks to configure this resource are included in the "Setup Demo" Task.
