**************
* Background *
**************

The sql scripts in this subfolder provides additional identities that are bulk loaded into OrangeHRM.  These
files can be adde to the SERI 6.4 resource image.

****************
* To Configure *
****************

   1) Copy files located in config/demo/data/orangehrm/HealthCare into a directory in the resource image.
     For example: C:\SailPoint\SERI\data\HealthCare.

   2) Change directory to C:\SailPoint\SERI\data\HealthCare
   
   3) Execute the LoadHC.bat file from that directory
